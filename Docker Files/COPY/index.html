Welcome to Our Colorful Page

This is a sample `index.html` file demonstrating different ways to use colors with HTML and CSS. Colors can be specified using **color names**, **hex codes**, **RGB**, or **HSL** values.

This section uses internal CSS for its background color (HSLA) and has text with an inline CSS color name (Maroon).
